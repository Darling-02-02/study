# 术语表与指标速查（新人入门版）
**适用方向**：三维重建、产量预测、机器学习、光谱与品质预测

## 1. 常见术语（先看这个）
- `3D重建`：从多视角图像或深度数据恢复目标三维结构（点云/网格）。
- `点云`：由大量三维点组成的数据结构，可表示物体几何形状。
- `ROI`（Region of Interest）：感兴趣区域，建模时只取目标区域数据。
- `高光谱`（HSI）：每个像素包含连续多波段光谱信息，而不只是 RGB。
- `多光谱`：波段数少于高光谱，通常是从高光谱筛选关键波段后的工程化方案。
- `特征工程`：从原始数据提取可用于模型学习的变量（如体积、均值谱、纹理）。
- `特征选择`：从全部特征中筛掉冗余或噪声变量，保留关键特征。
- `多模态融合`：把几何、光谱、图像等不同来源特征联合建模。
- `多任务学习`：一个模型同时预测多个目标（如产量+品质+等级）。
- `域偏移`：训练环境与真实应用环境分布不一致导致性能下降。
- `过拟合`：模型在训练集表现好，在新数据表现差。
- `数据泄漏`：训练时“提前看到了测试信息”，导致虚高结果。

## 2. 常见模型（一句话理解）
- `PLS/PLSR/PLS-DA`：化学计量学经典方法，小样本下常用，解释性较好。
- `RF`：随机森林，稳健、易用，常作为强基线。
- `GBDT/XGBoost/LightGBM`：树模型增强家族，通常在表格特征上效果好。
- `SVM/SVR`：小样本表现常不错，对参数较敏感。
- `RNN/1D-CNN`：适合一维光谱序列建模。
- `U-Net`：常用于分割任务（如根系图像分割）。

## 3. 回归指标速查（产量/品质预测最常用）
- `R2`（决定系数）
  - 含义：模型解释真实方差的能力。
  - 越大越好，理论上最高 1。
  - 经验：`>0.8` 通常较好；`0.5~0.8` 可用；具体看任务噪声上限。

- `RMSE`（均方根误差）
  - 含义：预测误差的平均量级（对大误差更敏感）。
  - 越小越好，单位与标签相同。

- `MAE`（平均绝对误差）
  - 含义：平均绝对偏差，直观。
  - 越小越好，对异常值不如 RMSE 敏感。

- `MAPE`（平均绝对百分比误差）
  - 含义：百分比误差。
  - 越小越好，但当真实值接近 0 时不稳定。

- `Rp` / `Rp2`（预测集相关系数/决定系数）
  - 常见于光谱文献，用于强调在预测集上的拟合能力。

- `RPD`
  - 含义：样本标准差与预测误差之比，常见于光谱定量。
  - 经验：`<1.5` 较差，`1.5~2` 一般，`>2` 可用，`>2.5` 较好。

## 4. 分类指标速查（成熟度/等级/异常识别）
- `Accuracy`：总体正确率，类别不平衡时可能误导。
- `Precision`：预测为正中有多少是真的正。
- `Recall`：真实为正中有多少被找出来。
- `F1-score`：Precision 与 Recall 的平衡指标。
- `Confusion Matrix`：最重要的诊断图，能看出哪一类最容易混淆。
- `AUC`：二分类常用，衡量整体区分能力。

## 5. 光谱预处理速查
- `SG`：平滑去噪，保留曲线形状。
- `MSC`：校正散射和基线偏移。
- `SNV`：标准正态变换，削弱散射差异。
- `一阶/二阶导`：增强峰形变化，但会放大噪声，需配合平滑。

## 6. 新人最常见错误与修正
- 错误 1：随机划分样本，导致同批次同时进训练和测试。
  - 修正：按批次/日期/温室区块做分组划分。

- 错误 2：只报训练集指标。
  - 修正：必须同时报告验证集和外部测试集指标。

- 错误 3：只看 Accuracy，不看混淆矩阵。
  - 修正：分类任务必须给出每类 Precision/Recall/F1。

- 错误 4：上来就用复杂深度模型。
  - 修正：先做 RF/GBDT/PLS/SVM 强基线，再迭代复杂模型。

- 错误 5：没有统一数据版本与样本 ID。
  - 修正：建立固定样本清单（ID、采集时间、设备参数、标签版本）。

## 7. 推荐的最小报告模板
1. 数据说明：样本量、类别分布、批次来源、划分方式。  
2. 预处理：使用了哪些方法，参数是什么。  
3. 模型：基线模型与改进模型列表。  
4. 指标：回归与分类分别报告。  
5. 可解释性：关键特征/关键波段。  
6. 泛化测试：外部测试集结果。  
7. 失败案例：至少 3 类典型错误样本分析。  

## 8. 结合你课题的实践建议
- 先建立“3D-only”和“HSI-only”两套可复现基线；
- 再做“3D+HSI 融合”消融实验；
- 每次实验固定一个变量变化，避免结论混淆；
- 先保证可解释和稳定，再追求最高分数。
